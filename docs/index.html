Features

Built with modern CSS features and best practices for optimal performance and maintainability.

Includes TypeScript components for enhanced development experience and type safety.

Responsive Design

Getting Started

Installation

Install using npm:

npm install codelace

Or using yarn:

yarn add codelace

Usage

Import the CSS:

// Using ES modules
import 'codelace/css';

// Using CommonJS
require('codelace/css');

Import TypeScript components:

// Import specific components
import { Modal, Button, Dropdown } from 'codelace';

// Use in your code
const modal = new Modal('#myModal');
modal.show();
