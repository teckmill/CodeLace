Features

Everything you need to build modern web applications

Modern CSS Grid

Built with CSS Grid and Flexbox for powerful, responsive layouts. Includes predefined grid systems and flexible containers.

TypeScript Components

First-class TypeScript support with interactive components. Includes type definitions, event handling, and state management.

Responsive Design

Mobile-first approach with responsive breakpoints. Automatically adapts to different screen sizes and devices.

Customizable Theme

Easy customization with CSS variables. Change colors, typography, spacing, and more with simple variable overrides.

Performance First

Optimized for performance with minimal CSS output. Tree-shakeable components and utilities for production builds.

Developer Tools

Built-in developer tools for debugging and customization. Includes CSS inspection utilities and component playground.

Getting Started

Quick start guide to using CodeLace in your project

Installation

Install using npm:

npm install @codinglace/css

Or using yarn:

yarn add @codinglace/css

Usage

Import the CSS:

// Using ES modules import '@codinglace/css/css'; // Using CommonJS require('@codinglace/css/css');

Import TypeScript components:

// Import specific components import { Modal, Button, Dropdown } from '@codinglace/css';
