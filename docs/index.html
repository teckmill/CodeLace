Getting Started

Quick start guide to using CodeLace in your project

Installation

Install using npm:

npm install @teckmill/codelace

Or using yarn:

yarn add @teckmill/codelace

Usage

Import the CSS:

// Using ES modules import '@teckmill/codelace/css'; // Using CommonJS require('@teckmill/codelace/css');

Import and use TypeScript components:

// Import specific components import { Modal, Button, Dropdown } from '@teckmill/codelace'; // Initialize components const modal = new Modal('#myModal'); modal.show();
