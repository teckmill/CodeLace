## Getting Started

```bash
npm install codelace
```

## Usage

```typescript
import { Tab, Popover } from 'codelace';

// Initialize components
const tab = new Tab('#myTab');
const popover = new Popover('#myPopover', {
  content: 'Hello World!',
  placement: 'top'
});
```
